Azure AI learning path — beyond AI-900

AI-900 is the starting point. The Azure AI career path: AI-900 (Fundamentals) → AI-102 (Azure AI Engineer Associate) — builds and deploys AI solutions (₹9,000 exam fee, 3–4 months preparation, targets ₹8–15L roles) → AI-900 + Specialisations (Azure Cosmos DB, Azure OpenAI Service advanced usage). Most serious AI engineers pursue AI-102 after AI-900.

How to prepare for AI-900 — free resources

  • Microsoft Learn — official free learning path for AI-900 (8–10 hours total)
  • YouTube — "AI-900 full course" by Adam Marczak or John Savill (free, 4–6 hours)
  • Practice tests — MeasureUp (paid, ₹2,500) or free quizzes on Exam Topics
  • Hands-on — create a free Azure account, try Azure Vision API and Azure OpenAI playground
  • Study time: 15–20 hours preparation is sufficient for most backgrounds
